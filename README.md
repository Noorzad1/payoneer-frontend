# 💳 Payoneer Fullstack Project

این پروژه شامل دو بخش است:
- 🔧 **Backend (FastAPI)**
- 🌐 **Frontend (ReactJS)**

---

## 🧰 نصب و راه‌اندازی

### 1. اجرای Backend

```bash
cd backend
docker-compose up --build
```

### 2. اجرای Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 📦 تکنالوژی‌ها

- FastAPI, SQLAlchemy, bcrypt, JWT
- ReactJS, TailwindCSS
- Docker, Node.js
